# 🔄 SkillSwap — Skill Exchange Platform

> Students exchange skills — teach what you know, learn what you don't. No money needed.

## Live URLs
- **Frontend:** https://skillswap-two-woad.vercel.app
- **Backend:**  https://skillswap-backend-eerb.onrender.com

## Tech Stack
| Layer | Tech |
|-------|------|
| Frontend | React 18 + Vite + Tailwind CSS |
| Backend | Node.js + Express + Socket.io |
| Database | PostgreSQL (Render) |
| Realtime | WebRTC (calls) + Socket.io (messages) |
| Auth | JWT |
| Deploy | Vercel + Render |

## Features
- 👤 Auth — Register, Login, Edit profile
- 🔍 Browse — Search users by skill/category
- ✨ Smart Matches — Mutual skill exchange algorithm
- 🤝 Exchanges — Send/Accept/Complete swap requests + Reviews
- 💬 Real-time Chat — Live messages + typing indicators
- 📞 Voice Calls — WebRTC peer-to-peer audio
- 📹 Video Calls — WebRTC peer-to-peer video
- 🖥️ Screen Share — Share screen during video calls
- 🔔 Notifications — Real-time alerts

## Quick Start (Local)
```bash
# Backend
cd backend && npm install && npm run dev

# Frontend
cd frontend && npm install && npm run dev
```
